Hospital Performance Analysis Dashboard

## Project Overview
This project presents a comprehensive analysis of hospital performance using interactive data visualization techniques. 
The dashboard provides insights into key operational metrics such as revenue, patient volume, readmissions, and diagnosis patterns to support data-driven decision-making in healthcare management.

## Objectives
The main goals of this analysis include:
- Evaluating overall hospital performance using key metrics
- Analyzing patient trends and service utilization
- Identifying patterns in diagnoses and treatments
- Monitoring readmission rates to assess quality of care
- Supporting strategic planning through data insights

## Tools & Technologies
- Power BI– Dashboard creation and data visualization, Data cleaning and preprocessing  

## Key Metrics Analyzed
- Total Revenue  
- Total Patients  
- Total Procedures  
- Readmission Count  
- Diagnosis Distribution  
- Patient Demographics (Gender, Marital Status)  
- Geographic Distribution  

## Key Insights
- The hospital generated over **2 million in revenue**, indicating strong financial performance.
- Readmission cases highlight potential areas for improving patient care and follow-up processes.
- Certain diagnoses, such as movement therapy and combined disorders, occur more frequently than others.
- Patient distribution varies across locations, with some regions showing higher concentration.
- Trends in patient encounters suggest fluctuations over the years, which may be influenced by external factors such as policy changes or healthcare demand.

## Dataset
The dataset used for this project includes hospital records containing patient information, diagnoses, procedures, and financial data.  

## Conclusion
This analysis provides valuable insights into hospital operations and patient trends. The findings can help healthcare stakeholders make informed decisions to improve service delivery, optimize resource allocation, and enhance patient outcomes.
